   2 * # hsOsaES6

   3 * 

   4 * Helpful Scripts ES6 interface to various OSX functions using the OSA Architecture. 

   5 * "https://github.com/HelpfulScripts/hsOsaES6" target="_blank">[Github page]

   6 * "./data/src/hsOsaes6/coverage/" target="_blank">[Coverage Info]

   7 * ___

   8 * 

   9 * Provides a means for calling JavaScript functions in Apple's OSA environment. 

  10 * At the core, the function {@link osaLib.osaJS osaJS} is a port from [brandonhorst's node-osa](https://github.com/brandonhorst/node-osa) module,

  11 * in ES6 using Typescript and promises.

  12 * 

  13 * `osaJs` is internally called by a number of OSX convenience functions exported as 

  14 * {@link Commands.osaCommands osaCommands}.

  15 * Currently available functions are:

  16 * -   {@link Commands.osaCommands.facetime      facetime}      starts a facetime call with a buddy

  17 * -   {@link Commands.osaCommands.getEmail      getEmail}      gets recent emails from `Mail`

  18 * -   {@link Commands.osaCommands.sendEmail     sendEmail}     sends an email using `Mail`

  19 * -   {@link Commands.osaCommands.launch        launch}        launches an application (such as `Mail`)

  20 * -   {@link Commands.osaCommands.isRunning     isRunning}     checks if an application is running

  21 * -   {@link Commands.osaCommands.quit          quit}          quits a running application

  22 * -   {@link Commands.osaCommands.launchScript  launchScript}  launches another OSA script

  23 * -   {@link Commands.osaCommands.restart       restart}       restarts the Mac

  24 * -   {@link Commands.osaCommands.sendMessage   sendMessage}   sends a text message to a buddy via `Messages`

  25 * -   {@link Commands.osaCommands.setVolume     setVolume}     sets the speaker volume 

  26 * -   {@link Commands.osaCommands.say           say}           speaks a text string using the built-in voice synthesis

  27 * -   {@link Commands.osaCommands.setBrightness setBrightness} sets the screen brightness

  28 * 

  29 */
